Overview

BugRecon streamlines the reconnaissance phase of bug bounty hunting and penetration testing, allowing security researchers to focus on deeper analysis and exploitation.

Features

  • Target Input: Prompts for target domain.
  • Reconnaissance Depth: Customize scanning extent.
  • Subdomain Enumeration: Retrieves subdomains using crt.sh.
  • Port Scanning: Identifies open ports with nmap.
  • Directory Brute-Forcing: Discovers hidden directories with ffuf.
  • Vulnerability Scanning: Checks for SQLi and XSS.
  • Screenshot Capturing: Captures web app screenshots using EyeWitness.
  • Report Generation: Compiles findings into a report.
  • Modular Design: Easy to expand and integrate additional features.

Prerequisites

External Tools

  • Python 3.6+
  • Nmap: go install github.com/ffuf/ffuf@latest
  • ffuf: go install github.com/ffuf/ffuf@latest
  • EyeWitness: 
    git clone https://github.com/FortyNorthSecurity/EyeWitness.git
cd EyeWitness/Python/setup
python setup.py install

Python Packages

Install required packages using:

pip install -r requirements.txt

Installation

  1. Clone the Repository
  2. Install Python Dependencies
  3. Ensure External Tools are Installed
